# Tarot WebApp

This is a simple Tarot game made to learn ReactJS and Redux.

You can see it work at: https://marcoddm.github.io/tarot/

It is still under development :)

## Installation

**Install Node dependencies**
```sh
npm install
```

**Run the application**
```sh
npm run start
```

**The app will be running at**
```sh
http://localhost:3000
```
